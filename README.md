### WhatsApp MD user bot
### Setup

1. Click [SCAN](https://early-pie-production.up.railway.app/md) and scan QR through Whatsapp Linked Devices Option in Your whatsapp App.
2. You will get a session id in whatsapp, copy id only.
3. If You don't have a account in [Heroku](https://signup.heroku.com/), Create a account.
4. Click [FORK](https://github.com/lyfe00011/wa-bot/fork)
5. Now https://heroku.com/deploy?template=https://github.com/vishnukl29/wa-bot copy this url and change USERNAME with your github username and click GO<br>
